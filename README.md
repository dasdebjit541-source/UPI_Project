# UPI Transaction Analysis (2020–2026)

# Problem Statement
Analyze UPI transaction data to identify growth trends, seasonal patterns, and impact of COVID-19.

# Key Insights
- UPI transactions showed exponential growth post-2020  
- Peak transaction volume observed in *January 2026*  
- Strong seasonal spikes during *festive months (Sep–Oct)*  
- Temporary dip observed during COVID period followed by rapid recovery  

# Tools Used
- SQL (Aggregation, Time Analysis)
- Power BI (Dashboard Visualization)

 # HR Attrition Analysis

# Problem Statement
Identify key factors contributing to employee attrition in an organization.

# Key Insights
- Higher attrition observed in *low salary groups*  
- Employees aged *25–35* show maximum attrition  
- Certain departments have significantly higher turnover  
- Gender-based attrition difference is minimal  

# Tools Used
- SQL (Filtering, Grouping, Analysis)
- Power BI (Dashboard)
